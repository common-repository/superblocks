== Description ==

SuperBlocks is the best collection of blocks for the new Gutenberg editor. Create beautiful looking content with ease and without typing a single line of code.

Simply install the SuperBlocks plugin, create a new post in WordPress using the new Gutenberg editor, and add our amazing content blocks directly into your post. *It's that easy!*

== Installation ==

SuperBlocks requires WordPress Gutenberg to be installed. This will be the standard editor in WordPress 5.0. Until then, [download the Gutenberg editor as a plugin](https://wordpress.org/plugins/gutenberg/), and install it on your WordPress site.

1. Click the blue 'Download' button at the top of this page to download SuperBlocks
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Create a new post or page using Gutenberg
4. Click the button to add a new block to your post and you'll be able to see all of the blocks in the SuperBlocks collection!
